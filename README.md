Amazon Clone
📌 Project Overview

This is a responsive Amazon Clone homepage built using HTML5 and CSS3. The project recreates the look and feel of the Amazon homepage, including the navigation bar, search bar, hero section, shopping categories, and footer. It is designed to improve frontend development skills by practicing modern HTML and CSS concepts.

🚀 Features
Responsive Navigation Bar
Amazon-style Search Bar
Delivery Address Section
Sign In & Returns Section
Shopping Cart Icon
Navigation Panel
Hero Banner
Product Category Boxes
Hover Effects
Footer with Multiple Sections
Responsive Layout
🛠️ Technologies Used
HTML5
CSS3
Font Awesome Icons
